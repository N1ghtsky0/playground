# playground
재밌어 보이는걸 구현하고 공부한걸 기록하는 레포입니다.

# 실행 방법
***필수: 도커가 설치되어 있어야합니다.***
1. 레포지토리 클론
   ```text
   git clone https://github.com/N1ghtsky0/playground.git
   ``` 
   
2. 프로젝트 빌드
   ```text
   ./gradlew bootwar
   ```
   
3. 도커 컴포즈 실행
   ```text
   docker-compose up -d
   ```
   
4. 접속
   ```text
   http://localhost:8080/
   ```
